Visual Studio Code安装及插件推荐

系统：windows

一、VS Code安装

下载地址：https://code.visualstudio.com/Download

  • 现在的windows应该都是64位，选择64位下载即可

  • 安装时可以修改安装路径，最好不要再C盘安装，下图选项可以考虑勾选，打开文件和目录比较方便
