git安装与基础使用

Git是一个开源的分布式版本控制系统，可以有效、高速的处理从很小到非常大的项目版本管理。

一、git下载

https://git-scm.com/downloads

二、安装

  • 找到下载好的git.exe 文件，点击安装，设置安装路径（尽量不要装在C盘）

  • 一路点击 Next，当看到如下页面的时候，请选择第一项，表示使用Git Bash 编辑git命令

  • 继续 Next，看到如下界面时，选择第三项，避免在后续操作中报一些警告问题。接下来一路Next，直至安装完成即可。

三、git环境配置

1.环境变量

  • 在此电脑->属性->高级系统设置->环境变量打开环境变量, 在变量名为path的系统变量中添加你git的安装路径

  • 将git安装目录中的bin目录路径放进去

2.全局配置

  • 在任一文件夹中右击点击Git Bash Here

  • 依此输入下列命令 全局配置git的用户信息，作为连接远程仓库时的用户信息（每次推送远程的用户信息就是现在设置的用户信息），用户名可以是全拼，邮箱务必真实

3.生成公钥

  • 生成公钥，用于连接远程仓库的钥匙 注意邮箱与上一步设置的邮箱保持一致

  • 成功之后你会发现，在C盘，用户目录下多了一个 .ssh 文件夹， id_rsa.pub 存放的就是你的公钥

  • 公钥生成之后，必定是要连接远程仓库的，打开github，头像点击setting设置
  • 如果还没有github账号 先去github官网https://github.com/ 注册一下github

  • 进入setting之后，点击SSH and GPG keys选项，点击右上角New SSHkey 新增公钥

  • 将刚才生成的 id_rsa.pub 中的公钥复制到key中， Title自定义，点击 Add SSH key 新增成功

提交代码准备工作

  • 在github中点击new repository新建仓库

  • 新建后需要对项目进行初始化 会有命令提示

  • 去本地文件夹中选择你要推送的目录 右击Git Bash Here打开命令界面

  • 可以根据仓库后的命令提示进行初始化
